# Right Hire GoDaddy Site Package

This is a static website package designed to be easy to upload to GoDaddy hosting.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Before You Upload

Replace these placeholders in `index.html`:

- `contactus@myrighthire.com`
- `Add your main business number here`

Optional:

- swap the email CTA for a GoDaddy contact form
- add your logo assets
- add a favicon
- add analytics or CRM scripts

## Upload Steps

1. Log in to GoDaddy.
2. Open your hosting file manager or connect by FTP.
3. Go to the root public folder for `myrighthire.com`.
4. Back up the current site files first.
5. Upload `index.html`, `styles.css`, and `script.js`.
6. Refresh the site and test on desktop and mobile.

## Positioning Direction

This version is intentionally not a generic recruiting-firm site.

It positions:

- `Right Hire` as the search firm
- leadership and executive hiring in advanced tech and manufacturing as the focus
- `Meridian` as the recruiting and sourcing operating system
- `Vector` as the leadership decision layer
- candidate screening as one important capability, not the whole company story
